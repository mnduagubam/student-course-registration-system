# student-course-registration-system
## Description
A Java-based student enrollment system implementing an ADT to manage students and course registration (add/drop/display).

## Classes
- StudentsADT.java: defines the methods the Students class implements
- Course.java: represents a course that a student takes by using the name, section number, etc
- Students.java: represents the entire student body implemented as array and also holds the Student class
- StudentsDemo.java: tests the program by taking user input to determine which action to do

## How to Run
1. Clone this repository
2. Compile all the .java files
3. Run StudentsDemo.java (and follow the instructions in the console)
